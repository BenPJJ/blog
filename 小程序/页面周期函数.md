# 页面周期函数

## 1. 生命周期函数

### 监听页面加载

``` js
onLoad: funtion (options) {}
```

### 监听页面初次渲染完成

``` js
onReady: funtion () {}
```

### 监听页面显示

``` js
onShow: funtion () {}
```

### 监听页面隐藏

``` js
onHide: funtion () {}
```

### 监听页面卸载

``` js
onUnload: funtion () {}
```

## 2. 页面启动

一般页面启动，会首先调用这三个函数：

onLoad（页面加载，一个页面只会调用一次）

---》

onShow（页面显示，页面每次打开都会调用）

---》

onReady（页面完成初次渲染）
